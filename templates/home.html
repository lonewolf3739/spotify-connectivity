Back story & About this app

Spotify has started its services in India. I was trying to enjoy some music. I took random walks on Spotify Music Network to find good music. Guess what happened next? I did not listen to music, but built this app instead.

This app is largely inspired by Six degrees of separation. Spotify's Web APIs provide enough data to build some cool apps. Ideally there should be no bugs because I found there is only one connected component for this indian artists graph. If there are any bugs inform me @srikanth.chekuri92@gmail.com
